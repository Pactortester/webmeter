## WebMeter

A web api-performance tool based on jmeter.

## Installation

### default

```shell
pip install -U webmeter
```

### mirrors

```shell
pip install -i  https://mirrors.ustc.edu.cn/pypi/web/simple -U webmeter
```

## Quickstart

### default

```shell
python -m webmeter
```

### customize

```shell
python -m webmeter --host={ip} --port={port}
```
